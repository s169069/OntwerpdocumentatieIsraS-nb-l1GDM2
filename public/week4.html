LOST IN TRA(I)NSLATION

WEEK 4

Deze week kregen we uitleg van de docent over hoe we HTML en CSS kunnen gebruiken in Visual Studio Code. Eerlijk gezegd begreep ik in het begin niet veel. We leerden ook hoe we ChatGPT konden inzetten om code voor onze borden te genereren en deze in Visual Studio Code te gebruiken.

De HTML- en CSS-code die ChatGPT aanleverde, kwam voor een groot deel niet overeen met mijn ontwerp, waardoor ik moeite had met het aanpassen naar mijn eigen stijl vanuit Figma. Ondanks deze uitdagingen liet ik na de les mijn borden aan de docent zien voor feedback.

Over het algemeen waren de kleuren en animaties goed, maar er waren een paar punten om te verbeteren: de ruimte voor de treinstations moest ik aanpassen, zodat er voldoende plek is als er meer haltes bijkomen, en ik moest nadenken over hoe een 2D-bord samengaat met een 3D-trein. Deze opmerkingen neem ik zeker mee en ga ik de komende tijd verwerken.
